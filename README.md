GREYctrl is the worlds most advanced vapor device controller software with the first plugin manager.

Feature List (v0.2)

Core App
• Experimental startup warning gate before app use.
• Resizable window on both axes.
• Bottom tab navigation with full-tab click targets.
• Keyboard zoom controls for UI scale (cmd​/ctrl + and cmd​/ctrl -).
• Darkwave visual theme with animated starfield background.
• Animated GREY ​CTRL gradient title with live version display.

• Live connection state indicators.

Ready / Session Runtime
